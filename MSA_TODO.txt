  - IntelliJ Run Config의 Main class 패키지 오타 (notification vs notificationservice)
  - Kafka JSON Consumer의 spring.json.value.default.type 설정 누락
  - Super Type Token 패턴 학습

  - fetch() 비동기 race condition (페이지 navigation 으로 인해 in-flight 요청 취소)
  - await 로 완료 대기하도록 수정
  - navigator.sendBeacon / keepalive 등 대안 학습

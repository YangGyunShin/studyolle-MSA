처리 방침 개요

StudyOlle은 개인정보보호법(PIPA) 제30조에 따라 수집·이용하는 개인정보를 투명하게 관리합니다. 아래 테이블은 항목별 수집 목적, 보유 기간, 처리 방식을 명시합니다.

기술적·관리적 보안 조치
비밀번호 암호화

BCrypt 해시 알고리즘 적용. 비밀번호 원문은 저장하지 않으며 복호화가 불가능합니다.

CSRF 방어

Spring Security CSRF 토큰을 모든 POST 요청에 적용하여 CSRF 공격을 방어합니다.

접근 제어

역할 기반 접근 제어(RBAC). 관리자 페이지(/admin/**)는 ROLE_ADMIN 권한 필요.

토큰 갱신

Remember-Me 토큰은 접속마다 갱신됩니다. 토큰 탈취 감지 시 전체 세션 무효화.

정보 주체의 권리
  • 개인정보 열람 요구권 — 수집된 개인정보를 언제든지 확인할 수 있습니다.
  • 개인정보 정정·삭제 요구권 — 설정 페이지에서 직접 수정하거나 탈퇴를 통해 삭제할 수 있습니다.
  • 개인정보 처리 정지 요구권 — 서비스 이용 중단 요청이 가능합니다.
  • 자동화된 의사결정 거부권 — 현재 자동화된 의사결정에 개인정보를 활용하지 않습니다.

개인정보 관련 문의: 관리자에게 직접 연락하거나 서비스 내 문의 채널을 이용하세요.
